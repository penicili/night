Cara Memulai Bermain

Ikuti panduan mudah 4 langkah ini untuk bergabung ke dalam dunia Night Network dan mulai petualangan pertamamu.

1

Salin IP Server

Salin IP play.nightnetwork.biz.id:20066 ke clipboard-mu.

2

Buka Minecraft

Buka game Minecraft (bisa pakai Java Edition atau Bedrock Edition dari HP).

3

Tambah Server

Pilih menu "Multiplayer" lalu "Add Server". Tempelkan IP yang tadi disalin, lalu Join.

4

Register & Login

Ketik /register <password> <password> di chat. Jika sudah pernah main, gunakan /login <password>.

Pertanyaan Sering Diajukan

Bingung tentang suatu hal? Temukan jawabannya di sini.

Apakah server ini gratis untuk dimainkan?

Tentu saja! Bermain di Night Network adalah 100% gratis untuk semua orang. Jika Anda ingin mendukung operasional server, Anda bisa membeli paket Donatur/Rank di Web Store kami.

Versi Minecraft berapa yang digunakan?

Kami sangat menyarankan Anda menggunakan Minecraft versi 1.20.x untuk fitur yang paling optimal. Namun, Anda juga dapat mencoba masuk menggunakan beberapa versi di bawah atau di atasnya.

Apakah saya bisa main lewat HP (Bedrock)?

Ya! Server kami mendukung Cross-play. Pemain dari HP (Android/iOS) atau PC Windows 10/11 bisa bergabung menggunakan IP: iron1.zyroxahost.icu dengan Port: 19116.

Bagaimana cara menghubungi staff jika ada masalah?

Cara termudah dan tercepat untuk mendapatkan bantuan adalah dengan bergabung ke server Discord resmi kami dan membuat tiket (Open Ticket). Tim Staff akan merespons secepat mungkin.
